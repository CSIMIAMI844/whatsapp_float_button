WhatsApp Float Button

Botón flotante de WhatsApp para Odoo 18

Este módulo agrega un botón flotante de WhatsApp en la esquina inferior derecha del sitio web de Odoo. Es totalmente configurable desde Ajustes → Configuración de WhatsApp.

Características

  • Botón flotante visible en todas las páginas del sitio web.
  • Compatible con escritorio y móviles.
  • Configuración del número desde Ajustes.

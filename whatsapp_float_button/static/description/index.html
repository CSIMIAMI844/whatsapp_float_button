WhatsApp Float Button

Boton flotante de WhatsApp para Odoo 18

Este modulo agrega un boton flotante de WhatsApp en la esquina inferior derecha del sitio web de Odoo. Es totalmente configurable desde Ajustes - Configuracion de WhatsApp.

Caracteristicas

  • Boton flotante visible en todas las paginas del sitio web.
  • Compatible con escritorio y moviles.
  • Configuracion del numero desde Ajustes.
